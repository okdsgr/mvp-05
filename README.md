# アキねもん（AKINEMON）
	
	> おまかせあれ！たぶん当たる！
	
	![Godot](https://img.shields.io/badge/Godot-4.6.1-blue) ![Status](https://img.shields.io/badge/Status-WIP-yellow)
	
	## コンセプト
	
	**「何が何に勝つか」を、自信満々でよく間違えるAIと一緒に遊ぶターン制対戦ゲーム。**
	
	アキネーター×ドラえもんを掛け合わせた「アキねもん」が、あなたの出したいものをなんとか当てようとする。
	正確に当たることもあれば、微妙に外れることもある。でも外れたらそれはそれで面白い。
	
	## ゲームフロー
	
	```
	① CPU がエンティティを生成（例：カエル）
		↓ 画面上部に出現、3分かけてゆっくり降下
	② プレイヤーが「勝てるもの」を入力（例：ヘビ）
		↓ Claude が勝敗判定＋演出テキストを生成
	③ 勝者が残る → CPUが次のカウンターを出す
		↓ 例：マングース（ヘビに勝てるから）
	④ これを繰り返す
	```
	
	## 特徴
	
	| 要素 | 内容 |
	|------|------|
	| バトル判定 | Claude Haiku がリアルタイム判定 |
	| 演出テキスト | 判定と同時に3行の演出を自動生成 |
	| CPUカウンター | 毎ラウンド Claude が「勝者に勝てるもの」を考える |
	| 降下タイマー | 3分以内に入力しないと負け |
